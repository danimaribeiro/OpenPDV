Comunidade

Requisitos para desenvolver o OpenSIG e OpeNFE:

  • Java (recomendamos 1.6)
  • GWT (recomendamos 2.4)
  • Eclipse Indigo (recomendamos 3.7)
  • Banco de Dados (recomendamos MySQL 5.5)
  • Web Server (recomendamos Tomcat 7.0)
  • Fontes


Requisitos para desenvolver o OpenPDV e os sub-sistemas utilizados (OpenPAF, OpenECF e OpenTEF)

  • Java (recomendamos 1.7)
  • Netbeans (recomendamos 7.2)
  • Banco de Dados (recomendamos H2)
  • Fontes
    • OpenPDV (vai precisar baixas os sub-projetos abaixos e gerar os jars de cada)
    • OpenPAF
    • OpenECF
    • OpenTEF
Requisitos para utilizar o OpenPDV em modo de desenvolvimento
  • Emulador ECF (Serve para 90% dos testes, mas para homologar refaça os testes com uma ECF real)
  • Emulador Serial (Utilizado para emular COM e fechar uma comunicação entre elas)
  • Emuladores TEF (Os mesmos usados na certificação, quando for em produção serão outros)
    • Pay&Go
    • Gerenciador Padrão 4
      • Módulo Visanet
      • Módulo Redecard
      • Módulo Amex
